                    TOUR EIFFEL BATMAN MAP - FLUTTER APP
================================================================================

DESCRIPTION
-----------
Application Flutter multi-plateformes affichant une carte OpenStreetMap 
interactive centrée sur la Tour Eiffel à Paris, avec un overlay du logo 
Batman positionné au-dessus du monument.

PLATEFORMES SUPPORTÉES
----------------------
✓ Android (API 21+)
✓ iOS (iOS 12+)
✓ Web (Chrome, Firefox, Safari, Edge)

FONCTIONNALITÉS PRINCIPALES
----------------------------
• Carte interactive OpenStreetMap
• Centrage automatique sur la Tour Eiffel (48.858370, 2.294481)
• Overlay personnalisé avec logo Batman (opacité 70%)
• Marqueur rouge sur la Tour Eiffel
• Contrôles de zoom interactifs (+/-)
• Bouton de recentrage rapide
• Navigation par glissement tactile

TECHNOLOGIES UTILISÉES
----------------------
• Flutter SDK 3.8.1+
• Dart Language
• flutter_map 7.0.2 (Cartographie)
• latlong2 0.9.1 (Coordonnées géographiques)
• OpenStreetMap (Fournisseur de tuiles)

COMMANDES RAPIDES
-----------------
Installation :
  $ flutter pub get

Lancement Android :
  $ flutter run -d android

Lancement iOS :
  $ flutter run -d ios

Lancement Web :
  $ flutter run -d chrome

Script interactif :
  $ ./run.sh

Build Release Android :
  $ flutter build apk --release

Build Release iOS :
  $ flutter build ios --release

Build Web :
  $ flutter build web --release
